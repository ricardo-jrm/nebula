### <b>Bootstrap</b>

```ts
npx @ricardo-jrm/nebula
```

Boilerplate `Next` web app bootstrapped with all necessary configs and workflows

<br />

### <b>Usage</b>

```ts
// install deps
yarn install

// run dev server
yarn dev

// build app
yarn build

// start app
yarn start

// export app
yarn export
```
